# 🐙 横琴半导体新闻日报

每日横琴半导体产业新闻，由 AI 自动采集、分析、排版、评级。

## 内容

- **每日新闻**：采集自微信公众号、政策网站、RSS 源
- **AI 分析**：DeepSeek 大模型分析每篇文章的相关性和重要性
- **分类归档**：按人才/投资/政策/行业动态分类
- **同话题折叠**：相同话题的报道自动合并折叠
- **低相关过滤**：AI 自动过滤与横琴半导体无关的泛新闻

## 访问

打开 `index.html` 自动跳转到最新日报。
打开 `archive.html` 查看历史日报列表。

> 由 GitHub Actions + 本地 launchd 定时生成。
